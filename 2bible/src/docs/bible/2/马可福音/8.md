<a id="1"></a>8:1  那时，又有许多人聚集，并没有甚么吃的。耶稣叫门徒来，说：  

<a id="2"></a>8:2  “我怜悯这众人，因为他们同我在这里已经三天，也没有吃的了。  

<a id="3"></a>8:3  我若打发他们饿著回家，就必在路上困乏，因为其中有从远处来的。”  

<a id="4"></a>8:4  门徒回答说：“在这野地，从哪里能得饼，叫这些人吃饱呢？”  

<a id="5"></a>8:5  耶稣问他们说：“你们有多少饼？”他们说：“七个。”  

<a id="6"></a>8:6  他吩咐众人坐在地上，就拿著这七个饼祝谢了，擘开，递给门徒，叫他们摆开，门徒就摆在众人面前。  

<a id="7"></a>8:7  又有几条小鱼，耶稣祝了福，就吩咐也摆在众人面前。  

<a id="8"></a>8:8  众人都吃，并且吃饱了；收拾剩下的零碎，有七筐子。  

<a id="9"></a>8:9  人数约有四千。耶稣打发他们走了，  

<a id="10"></a>8:10  随即同门徒上船，来到大玛努他境内。  

<a id="11"></a>8:11  法利赛人出来盘问耶稣，求他从天上显个神迹给他们看，想要试探他。  

<a id="12"></a>8:12  耶稣心里深深地叹息，说：“这世代为甚么求神迹呢？我实在告诉你们，没有神迹给这世代看！”　  

<a id="13"></a>8:13  他就离开他们，又上船往海那边去了。  

<a id="14"></a>8:14  门徒忘了带饼，在船上除了一个饼，没有别的食物。  

<a id="15"></a>8:15  耶稣嘱咐他们说：“你们要谨慎，防备法利赛人的酵和希律的酵。”  

<a id="16"></a>8:16  他们彼此议论说：“这是因为我们没有饼吧？”  

<a id="17"></a>8:17  耶稣看出来，就说：“你们为甚么因为没有饼就议论呢？你们还不省悟，还不明白吗？你们的心还是愚顽吗？  

<a id="18"></a>8:18  你们有眼睛，看不见吗？有耳朵，听不见吗？也不记得吗？  

<a id="19"></a>8:19  我擘开那五个饼分给五千人，你们收拾的零碎装满了多少篮子呢？”他们说：“十二个。”  

<a id="20"></a>8:20  “又擘开那七个饼分给四千人，你们收拾的零碎装满了多少筐子呢？ ”他们说：“七个。”  

<a id="21"></a>8:21  耶稣说：“你们还是不明白吗？”  

<a id="22"></a>8:22  他们来到伯赛大，有人带一个瞎子来，求耶稣摸他。  

<a id="23"></a>8:23  耶稣拉著瞎子的手，领他到村外，就吐唾沫在他眼睛上，按手在他身上，问他说：“你看见甚么了？”  

<a id="24"></a>8:24  他就抬头一看，说：“我看见人了，他们好像树木，并且行走。”  

<a id="25"></a>8:25  随后又按手在他眼睛上，他定睛一看，就复了原，样样都看得清楚了。  

<a id="26"></a>8:26  耶稣打发他回家，说：“连这村子你也不要进去。”  

<a id="27"></a>8:27  耶稣和门徒出去，往凯撒利亚腓立比的村庄去。在路上问门徒说：“ 人说我是谁？”  

<a id="28"></a>8:28  他们说：“有人说是施洗的约翰，有人说是以利亚，又有人说是先知里的一位。”  

<a id="29"></a>8:29  又问他们说：“你们说我是谁？”彼得回答说：“你是基督。”  

<a id="30"></a>8:30  耶稣就禁戒他们，不要告诉人。  

<a id="31"></a>8:31  从此，他教训他们说：“人子必须受许多的苦，被长老、祭司长和文士弃绝，并且被杀，过三天复活。”  

<a id="32"></a>8:32  耶稣明明地说这话，彼得就拉著他劝他。  

<a id="33"></a>8:33  耶稣转过来，看著门徒，就责备彼得说：“撒但，退我后边去吧！因为你不体贴 神的意思，只体贴人的意思。”  

<a id="34"></a>8:34  于是叫众人和门徒来，对他们说：“若有人要跟从我，就当舍己，背起他的十字架来跟从我。  

<a id="35"></a>8:35  因为，凡要救自己生命（注：或作“灵魂”。下同）的，必丧掉生命；凡为我和福音丧掉生命的，必救了生命。  

<a id="36"></a>8:36  人就是赚得全世界，赔上自己的生命，有甚么益处呢？  

<a id="37"></a>8:37  人还能拿甚么换生命呢？  

<a id="38"></a>8:38  凡在这淫乱罪恶的世代，把我和我的道当作可耻的，人子在他父的荣耀里，同圣天使降临的时候，也要把那人当作可耻的。”
