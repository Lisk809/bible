<a id="1"></a>6:1  约伯回答说：  

<a id="2"></a>6:2  “惟愿我的烦恼称一称，我一切的灾害放在天平里，  

<a id="3"></a>6:3  现今都比海沙更重，所以我的言语急躁。  

<a id="4"></a>6:4  因全能者的箭射入我身，其毒，我的灵喝尽了；　神的惊吓摆阵攻击我。  

<a id="5"></a>6:5  野驴有草岂能叫唤？牛有料岂能吼叫？  

<a id="6"></a>6:6  物淡而无盐岂可吃吗？蛋青有甚么滋味呢？  

<a id="7"></a>6:7  看为可厌的食物，我心不肯挨近。  

<a id="8"></a>6:8  惟愿我得著所求的，愿　神赐我所切望的──  

<a id="9"></a>6:9  就是愿　神把我压碎，伸手将我剪除。  

<a id="10"></a>6:10  我因没有违弃那圣者的言语，就仍以此为安慰，在不止息的痛苦中还可踊跃。  

<a id="11"></a>6:11  我有甚么气力使我等候？我有甚么结局使我忍耐？  

<a id="12"></a>6:12  我的气力岂是石头的气力？我的肉身岂是铜的呢？  

<a id="13"></a>6:13  在我岂不是毫无帮助吗？智慧岂不是从我心中赶出净尽吗？  

<a id="14"></a>6:14  那将要灰心、离弃全能者、不敬畏　神的人，他的朋友当以慈爱待他。  

<a id="15"></a>6:15  我的弟兄诡诈，好像溪水，又像溪水流乾的河道。  

<a id="16"></a>6:16  这河，因结冰发黑，有雪藏在其中；  

<a id="17"></a>6:17  天气渐暖，就随时消化；日头炎热，便从原处乾涸。  

<a id="18"></a>6:18  结伴的客旅离弃大道，顺河边行，到荒野之地死亡。  

<a id="19"></a>6:19  提玛结伴的客旅瞻望，示巴同夥的人等候。  

<a id="20"></a>6:20  他们因失了盼望就抱愧，来到那里便蒙羞。  

<a id="21"></a>6:21  现在你们正是这样，看见惊吓的事便惧怕。  

<a id="22"></a>6:22  我岂说，‘请你们供给我，从你们的财物中送礼物给我’？  

<a id="23"></a>6:23  岂说，‘拯救我脱离敌人的手’吗？‘救赎我脱离强暴人的手’吗？  

<a id="24"></a>6:24  请你们教导我，我便不作声；使我明白在何事上有错。  

<a id="25"></a>6:25  正直的言语力量何其大！但你们责备，是责备甚么呢？  

<a id="26"></a>6:26  绝望人的讲论既然如风，你们还想要驳正言语吗？  

<a id="27"></a>6:27  你们想为孤儿拈阄，以朋友当货物。  

<a id="28"></a>6:28  现在请你们看看我，我决不当面说谎。  

<a id="29"></a>6:29  请你们转意，不要不公；请再转意，我的事有理。  

<a id="30"></a>6:30  我的舌上，岂有不义吗？我的口里，岂不辨奸恶吗？
