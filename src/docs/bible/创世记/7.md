<a id="1"></a>7:1  耶和华对挪亚说：“你和你的全家都要进入方舟，因为在这世代中，我见你在我面前是义人。　  

<a id="2"></a>7:2  凡洁净的畜类，你要带七公七母；不洁净的畜类，你要带一公一母；  

<a id="3"></a>7:3  空中的飞鸟也要带七公七母，可以留种，活在全地上；　  

<a id="4"></a>7:4  因为再过七天，我要降雨在地上四十昼夜，把我所造的各种活物都从地上除灭。”  

<a id="5"></a>7:5  挪亚就遵著耶和华所吩咐的行了。  

<a id="6"></a>7:6  当洪水泛滥在地上的时候，挪亚整六百岁。  

<a id="7"></a>7:7  挪亚就同他的妻和儿子、儿妇，都进入方舟，躲避洪水。  

<a id="8"></a>7:8  洁净的畜类和不洁净的畜类，飞鸟并地上一切的昆虫，  

<a id="9"></a>7:9  都是一对一对地，有公有母，到挪亚那里进入方舟，正如　神所吩咐挪亚的。  

<a id="10"></a>7:10  过了那七天，洪水泛滥在地上。  

<a id="11"></a>7:11  当挪亚六百岁，二月十七日那一天，大渊的泉源都裂开了，天上的窗户也敞开了。  

<a id="12"></a>7:12  四十昼夜降大雨在地上。  

<a id="13"></a>7:13  正当那日，挪亚和他三个儿子闪、含、雅弗，并挪亚的妻子和三个儿妇，都进入方舟。  

<a id="14"></a>7:14  他们和百兽，各从其类；一切牲畜，各从其类；爬在地上的昆虫，各从其类；一切禽鸟，各从其类，都进入方舟。  

<a id="15"></a>7:15  凡有血肉、有气息的活物，都一对一对地到挪亚那里，进入方舟。  

<a id="16"></a>7:16  凡有血肉进入方舟的，都是有公有母，正如　神所吩咐挪亚的。耶和华就把他关在方舟里头。  

<a id="17"></a>7:17  洪水泛滥在地上四十天，水往上长，把方舟从地上漂起。  

<a id="18"></a>7:18  水势浩大，在地上大大地往上长，方舟在水面上漂来漂去。  

<a id="19"></a>7:19  水势在地上极其浩大，天下的高山都淹没了。  

<a id="20"></a>7:20  水势比山高过十五肘，山岭都淹没了。  

<a id="21"></a>7:21  凡在地上有血肉的动物，就是飞鸟、牲畜、走兽和爬在地上的昆虫，以及所有的人都死了；  

<a id="22"></a>7:22  凡在旱地上、鼻孔有气息的生灵都死了；  

<a id="23"></a>7:23  凡地上各类的活物，连人带牲畜、昆虫，以及空中的飞鸟，都从地上除灭了，只留下挪亚和那些与他同在方舟里的。  

<a id="24"></a>7:24  水势浩大，在地上共一百五十天。
