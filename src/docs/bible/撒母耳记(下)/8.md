<a id="1"></a>8:1  此后，大卫攻打非利士人，把他们治服，从他们手下夺取了京城的权柄（注：原文作“母城的嚼环”）。  

<a id="2"></a>8:2  又攻打摩押人，使他们躺卧在地上，用绳量一量，量二绳的杀了，量一绳的存留。摩押人就归服大卫，给他进贡。  

<a id="3"></a>8:3  琐巴王利合的儿子哈大底谢往大河去，要夺回他的国权。大卫就攻打他，  

<a id="4"></a>8:4  擒拿了他的马兵一千七百，步兵二万；将拉战车的马砍断蹄筋，但留下一百辆车的马。  

<a id="5"></a>8:5  大马士革的亚兰人来帮助琐巴王哈大底谢，大卫就杀了亚兰人二万二千。  

<a id="6"></a>8:6  于是，大卫在大马士革的亚兰地设立防营，亚兰人就归服他，给他进贡。大卫无论往哪里去，耶和华都使他得胜。  

<a id="7"></a>8:7  他夺了哈大底谢臣仆所拿的金盾牌，带到耶路撒冷。  

<a id="8"></a>8:8  大卫王又从属哈大底谢的比他和比罗他城中夺取了许多的铜。  

<a id="9"></a>8:9  哈马王陀以听见大卫杀败哈大底谢的全军，  

<a id="10"></a>8:10  就打发他儿子约兰去见大卫王，问他的安，为他祝福，因为他杀败了哈大底谢。（原来陀以与哈大底谢常常争战。）约兰带了金银铜的器皿来，  

<a id="11"></a>8:11  大卫王将这些器皿，和他治服各国所得来的金银都分别为圣，献给耶和华。  

<a id="12"></a>8:12  就是从亚兰，摩押，亚扪，非利士，亚玛力人所得来的，以及从琐巴王利合的儿子哈大底谢所掠之物。  

<a id="13"></a>8:13  大卫在盐谷击杀了亚兰（注：或作“以东”，见诗篇60篇诗题）一万八千人回来，就得了大名。  

<a id="14"></a>8:14  又在以东全地设立防营，以东人就都归服大卫。大卫无论往哪里去，耶和华都使他得胜。  

<a id="15"></a>8:15  大卫作以色列众人的王，又向众民秉公行义。  

<a id="16"></a>8:16  洗鲁雅的儿子约押作元帅；亚希律的儿子约沙法作史官；　  

<a id="17"></a>8:17  亚希突的儿子撒督和亚比亚他的儿子亚希米勒作祭司长；西莱雅作书记；  

<a id="18"></a>8:18  耶何耶大的儿子比拿雅统辖基利提人和比利提人。大卫的众子都作领袖。
