<a id="1"></a>17:1  论大马士革的默示：“看哪，大马士革已被废弃，不再为城，必变作乱堆！  

<a id="2"></a>17:2  亚罗珥的城邑已被撇弃，必成为牧羊之处；羊在那里躺卧，无人惊吓。  

<a id="3"></a>17:3  以法莲不再有保障；大马士革不再有国权；亚兰所剩下的，必像以色列人的荣耀消灭一样。”这是万军之耶和华说的。  

<a id="4"></a>17:4  “到那日，雅各的荣耀必至枵薄，他肥胖的身体必渐瘦弱；  

<a id="5"></a>17:5  就必像收割的人收敛禾稼，用手割取穗子，又像人在利乏音谷拾取遗落的穗子。  

<a id="6"></a>17:6  其间所剩下的不多，好像人打橄榄树，在尽上的枝梢上只剩两三个果子，在多果树的旁枝上只剩四五个果子。”这是耶和华以色列的　神说的。  

<a id="7"></a>17:7  当那日，人必仰望造他们的主，眼目重看以色列的圣者。  

<a id="8"></a>17:8  他们必不仰望祭坛，就是自己手所筑的，也不重看自己指头所做的，无论是木偶是日像。  

<a id="9"></a>17:9  在那日，他们的坚固城必像树林中和山顶上所撇弃的地方，就是从前在以色列人面前被人撇弃的。这样，地就荒凉了。  

<a id="10"></a>17:10  因你忘记救你的　神，不记念你能力的磐石，所以，你栽上佳美的树秧子，插上异样的栽子。  

<a id="11"></a>17:11  栽种的日子，你周围圈上篱笆，又到早晨使你所种的开花；但在愁苦极其伤痛的日子，所收割的都飞去了。  

<a id="12"></a>17:12  唉！多民哄嚷，好像海浪□訇；列邦奔腾，好像猛水滔滔；  

<a id="13"></a>17:13  列邦奔腾，好像多水滔滔；但　神斥责他们，他们就远远逃避，又被追赶，如同山上的风前糠，又如暴风前的旋风土。  

<a id="14"></a>17:14  到晚上有惊吓，未到早晨他们就没有了。这是掳掠我们之人所得的分，是抢夺我们之人的报应！
