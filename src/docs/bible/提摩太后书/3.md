<a id="1"></a>3:1  你该知道，末世必有危险的日子来到。  

<a id="2"></a>3:2  因为那时人要专顾自己、贪爱钱财、自夸、狂傲、谤□、违背父母、忘恩负义、心不圣洁、  

<a id="3"></a>3:3  无亲情、不解怨、好说谗言、不能自约、性情凶暴、不爱良善、  

<a id="4"></a>3:4  卖主卖友、任意妄为、自高自大、爱宴乐、不爱　神，  

<a id="5"></a>3:5  有敬虔的外貌，却背了敬虔的实意，这等人你要躲开。  

<a id="6"></a>3:6  那偷进人家，牢笼无知妇女的，正是这等人。这些妇女担负罪恶，被各样的私欲引诱，  

<a id="7"></a>3:7  常常学习，终久不能明白真道。  

<a id="8"></a>3:8  从前雅尼和佯庇怎样敌挡摩西，这等人也怎样敌挡真道。他们的心地坏了，在真道上是可废弃的。  

<a id="9"></a>3:9  然而他们不能再这样敌挡，因为他们的愚昧必在众人面前显露出来，像那二人一样。  

<a id="10"></a>3:10  但你已经服从了我的教训、品行、志向、信心、宽容、爱心、忍耐，  

<a id="11"></a>3:11  以及我在安提阿、以哥念、路司得所遭遇的逼迫、苦难。我所忍受是何等的逼迫！但从这一切苦难中，主都把我救出来了。  

<a id="12"></a>3:12  不但如此，凡立志在基督耶稣里敬虔度日的，也都要受逼迫。  

<a id="13"></a>3:13  只是作恶的和迷惑人的，必越久越恶，他欺哄人，也被人欺哄。  

<a id="14"></a>3:14  但你所学习的、所确信的，要存在心里，因为你知道是跟谁学的。  

<a id="15"></a>3:15  并且知道你是从小明白圣经；这圣经能使你因信基督耶稣有得救的智慧。  

<a id="16"></a>3:16  圣经都是　神所默示的（注：或作“凡　神所默示的圣经”），于教训、督责、使人归正、教导人学义都是有益的，  

<a id="17"></a>3:17  叫属　神的人得以完全，预备行各样的善事。
