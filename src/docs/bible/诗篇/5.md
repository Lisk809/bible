<a id="1"></a>5:1  耶和华啊，求你留心听我的言语，顾念我的心思。  

<a id="2"></a>5:2  我的王我的　神啊，求你垂听我呼求的声音，因为我向你祈祷！  

<a id="3"></a>5:3  耶和华啊，早晨你必听我的声音；早晨我必向你陈明我的心意，并要警醒。  

<a id="4"></a>5:4  因为你不是喜悦恶事的　神，恶人不能与你同居。  

<a id="5"></a>5:5  狂傲人不能站在你眼前；凡作孽的，都是你所恨恶的。  

<a id="6"></a>5:6  说谎言的，你必灭绝；好流人血弄诡诈的，都为耶和华所憎恶。  

<a id="7"></a>5:7  至于我，我必凭你丰盛的慈爱进入你的居所；我必存敬畏你的心向你的圣殿下拜。  

<a id="8"></a>5:8  耶和华啊，求你因我的仇敌，凭你的公义引领我，使你的道路在我面前正直。  

<a id="9"></a>5:9  因为他们的口中没有诚实；他们的心里满有邪恶。他们的喉咙是敞开的坟墓；他们用舌头谄媚人。  

<a id="10"></a>5:10  神啊，求你定他们的罪。愿他们因自己的计谋跌倒；愿你在他们许多的过犯中把他们逐出，因为他们背叛了你。  

<a id="11"></a>5:11  凡投靠你的，愿他们喜乐，时常欢呼，因为你护庇他们；又愿那爱你名的人都靠你欢欣；  

<a id="12"></a>5:12  因为你必赐福与义人。耶和华啊，你必用恩惠如同盾牌四面护卫他。〔大卫的诗，交与伶长。用丝弦的乐器，调用第八。〕
