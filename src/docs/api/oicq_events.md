# oicq v2 标准事件 {#oicq_v2-events}

要查看事件详情、回调参数等，请参阅 `oicq` v2 文档的 [`EventMap`](https://oicqjs.github.io/oicq/interfaces/EventMap.html) 定义。

> 可以用插件 API [`plugin.on`](/api/plugin#plugin-on) 监听和处理 oicq 事件。
